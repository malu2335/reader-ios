# 隐私声明

生效日期：2026-07-18
适用版本：1.3.x
App：纸羽轻阅
开发者：Lu Ma（malu2335）
联系邮箱：malu233333@gmail.com

> 纸羽轻阅是一款本地优先的阅读器：导入、阅读、书架与备份均在本机完成；仅当您主动开启并配置「AI 翻译」时，您选中的文本才会发送到您自己配置的第三方 AI 服务。本声明说明 App 如何处理与您相关的信息，并与应用内实际行为、PrivacyInfo.xcprivacy 及 App Store Connect「App 隐私」问卷保持一致。本文不构成法律意见。

## 一、产品定位

纸羽轻阅专注于您自行导入的文件，在本机完成阅读。当前支持：

- TXT
- EPUB
- MOBI / AZW
- PDF
- ZIP / CBZ（漫画压缩包）

App 不会主动从网络获取书籍内容，也不提供账号、书城或云端同步。App 内置可选的「AI 翻译」功能，说明见第四、七节。

## 二、我们收集哪些信息

根据当前产品设计与依赖审计：

- 无账号注册或登录
- 无广告 SDK
- 无第三方分析或统计 SDK
- 无追踪（Tracking）；隐私清单中 NSPrivacyTracking 为 false
- 无开发者自有服务器；开发者不运营任何后端，不会收集、汇总或上传您的阅读内容或个人信息
- 隐私清单中 NSPrivacyCollectedDataTypes 为空
- 若您主动开启并配置「AI 翻译」，您选中的原文文本与您填写的服务商 API Key 会发送给您自己选择的第三方 AI 服务商（如 OpenAI、Anthropic、Gemini、小米 MiMo 或您自定义的兼容端点），详见第四、七节；开发者本身不经手、不留存这部分内容

因此，开发者不会通过本 App 收集您的姓名、邮箱、广告标识、阅读内容或书架列表并上传到开发者自有服务器。

> 上述结论以最终 Release 构建与依赖审计为准。若后续版本引入统计、账号能力或开发者自有服务器，将同步更新本声明与 App Store 隐私问卷。

## 三、保存在您设备上的数据

下列数据保存在设备上的 App 沙盒，或由您主动选择的备份位置：

- 导入的书籍副本与解析结果（章节、封面等）
- 阅读进度、书签与相关阅读记录
- 阅读设置（字体、主题、行距等）
- 自定义字体
- 正文净化 / 替换规则
- 朗读语音偏好
- 您主动创建的备份内容

数据受系统文件保护与设备安全能力保护。卸载 App 会删除沙盒内数据；您另存到「文件」、网盘或其他位置的备份，需自行管理与删除。

## 四、权限与系统能力

以下能力由 iOS 提供，并在您主动操作时使用：

- 文件访问 / 文档选择器：导入书籍与字体，或选择备份恢复；也可通过系统「用其他应用打开」传入文件
- 相册写入：在您主动保存时，将金句卡片或封面写入相册
- 系统词典：阅读中查词
- 系统朗读 / 个人声音：正文朗读与语音选择
- 系统分享面板：分享备份、摘录等内容到您选择的目标
- 后台音频：朗读时可在后台继续播放，不用于静默采集或上传
- 网络访问：仅用于「AI 翻译」——将您选中的文本发送到您自己配置的第三方 AI 服务商并接收译文；未开启或未配置该功能时，App 不会主动发起这类请求

App 不请求通讯录、位置、麦克风录音（非系统朗读）、相机等与阅读无关的权限。

## 五、数据分享与披露

开发者不会出售个人信息，也不会将阅读内容上传到开发者服务器。

仅在您主动操作或开启相关功能时，数据可能离开本 App：

- 开启「AI 翻译」后，选中文本发送给您自己配置的第三方 AI 服务商（详见第四、七节）
- 导出备份并通过系统分享发送到您选择的应用或位置
- 分享摘录、金句卡片或封面
- 使用系统词典、朗读、分享等能力时，由 iOS 或目标应用按其自身政策处理

开发者无法控制第三方如何处理您主动分享的内容，请仅向可信目标分享。

## 六、数据保留与删除

- 单本书：可从书架删除及其相关本地数据
- 全部书籍：可在设置中使用「清空书架」
- 卸载 App：删除沙盒内数据（书籍副本、数据库、设置等）
- 外部备份：保存在「文件」、网盘、电脑等处的文件需自行删除
- 相册中主动保存的图片：需在相册中自行删除

## 七、AI 翻译与安全说明

- App 不上传阅读内容到开发者服务器；开发者不运营任何后端
- 「AI 翻译」为可选功能，需您在设置中主动开启并填写服务商与 API Key 后才会生效；开启前 App 不会发起该类请求
- 开启后，您选中的原文文本会通过 HTTPS（或您自行配置的兼容端点）发送给您选择的第三方服务商用于生成译文；该服务商如何处理这些文本，以其自身隐私政策为准，开发者无法控制
- **在线朗读（HttpTTS / AI TTS）**：仅当您主动选择「在线朗读引擎」或「AI 模型朗读」时，待朗读的正文片段才会发送到您配置的第三方 TTS / AI 服务（含 OpenAI 兼容 /v1/audio/speech，或小米 MiMo chat 式语音合成）；默认使用系统本地语音，不外发文本
- API Key 仅保存在系统 Keychain；导出备份时密钥字段留空，不写入备份文件
- 除 AI 翻译外，App 不发起其他业务网络请求
- 请妥善保管设备解锁方式、AI 服务商 API Key，以及导出到外部的备份
- 不可信来源的文件可能含有异常内容；App 会对部分格式做资源预算与校验，但无法保证解析一切恶意构造文件
- 任何软件与设备都无法保证绝对安全

## 八、儿童隐私

本 App 面向一般阅读场景，不专门面向儿童收集个人信息。当前版本无账号、不收集个人信息；儿童在监护人指导下使用本地阅读时，也不会产生开发者侧的个人数据收集。监护人如需清理本地数据，可删除书籍或卸载 App。
